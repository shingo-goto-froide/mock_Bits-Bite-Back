# 企画者への確認事項

> プレイテスト（2026-04-09）で発生した疑問・モック実装時の判断をまとめたもの。
> 企画者に確認を取り、回答に応じて仕様書・設計書を更新する。

---

## 現在のモック実装での判断（要確認）

### 1. 魔物がやられた場合どうなるか
- **現在の実装**: バトル画面から非表示になる。墓守の能力で復活した場合は再表示
- **確認したいこと**: やられた魔物はどうなる想定か？（消える / グレーアウトで残る / 演出がある等）

### 2. 前の魔物がやられたら詰めるか
- **現在の実装**: 前方の味方が倒れると後方の味方が自動的に前進する（射程計算・表示ともに）
- **確認したいこと**: 前衛が倒れたら後衛が前に詰める想定で合っているか？

### 3. 出撃条件（1体以上 or 全枠必須）
- **現在の実装**: 隊列に1体以上配置すれば出撃可能。空きスロットはバトル時に詰める
- **確認したいこと**: 最低何体から出撃可能か？枠を埋めなくても出撃できる想定か？

### 4. 射程外の場合の行動
- **現在の実装**: 射程内に敵がいなければ攻撃をスキップする
- **確認したいこと**: 射程外の場合はスキップでよいか？それとも最も近い敵を攻撃する等の代替行動があるか？

## 企画書との差異（モック側で変更した点）

### 5. ガーディアンの枠数
- **企画書**: 編成例①③ではガーディアン含めて5体（6枠必要）→ 2枠ユニット想定
- **モック**: 5枠に収めるため **1枠に変更**
- **確認したいこと**: ガーディアンは1枠と2枠どちらが意図か？2枠の場合、最大枠数を6に増やすか？

### 6. ガーディアンの射程
- **企画書**: 射程2
- **モック**: 射程1のまま（変更していない）
- **確認したいこと**: 射程2が正しいか？

### 7. 初期素材でガーディアンが練成できない
- **現状**: 古い剣が初期2個、ガーディアン練成に3個必要 → 報酬で古い剣を入手するまで練成不可
- **確認したいこと**: 意図的か？初期素材を増やすべきか？

### 8. 分解の仕様
- **企画書**: 分解 → 魂を獲得（魂システムでパラメータ・特性カスタマイズ）
- **モック**: 魂システムはスコープ外のため、分解 → **素材を一部回収（50%）** に変更
- **確認したいこと**: モックの代替仕様で問題ないか？

### 9. スケルトンの攻撃力
- **企画書の編成例②**: ATK=3と記載
- **モック**: ATK=1（仕様書通り）
- **確認したいこと**: 編成例②のATK=3はランクシステムによる強化値か？それとも基本ATKが3が正しいか？

### 10. 殉教者の行動条件
- **現在の実装**: ダメージを受けている味方がいなければ行動しない（HP消費しない）
- **確認したいこと**: 回復対象がいない場合は何もしなくてよいか？

### 11. バトル後のHP0ユニットの扱い
- **現在の実装**: バトル終了後、HP0の魔物はHP1で復帰して準備画面に戻る
- **確認したいこと**: HP0で消滅 / HP1で復帰 / その他、どれが意図か？（モックではテスト効率のためHP1復帰を採用）

## 各魔物の能力に関する疑問（エッジケース）

### 12. スケルトン「1つ前の味方の攻撃力+1」
- 最前衛に配置した場合、前に味方がいない → 効果なしでよいか？
- バトルを重ねるごとにATK+1が累積する想定か？（企画書の編成例②ではATK=3で記載されており累積を示唆）

### 13. シェイプシフター「1つ後ろの魔物のステータスと特殊効果をコピー」
- **最後尾に配置して後ろに魔物がいない場合はどうなる？**（現在の実装: 何もしない）
- コピーは行動時のみで行動後は元に戻る想定か？（現在の実装: 行動後に元のステータスに戻す）
- 後ろの魔物が死亡している場合はコピーできるか？

### 14. レイス「死亡時に最後尾の魔物をHP+10、攻撃力+3」
- レイス自身が最後尾で死んだ場合、対象がいない → 効果なしでよいか？
- 最後尾の味方も死んでいる場合は？生存している最後尾を探すべきか？

### 15. 結界師「バトル開始時、前衛にダメージ半減の魔法バリア」
- 「前衛」は最前衛1体のみか？前衛数体か？（現在の実装: 最前衛1体のみ）
- 魔法バリアは物理攻撃には効果なしでよいか？（現在の実装: 魔法攻撃のみ半減）

### 16. 墓守「死亡時にこの階層で死んだ他の味方を全て復活」
- 復活時のHPは？（現在の実装: 最大HPの50%）
- 墓守が先に死に、その後別の味方が死んだ場合、その味方は復活しない？（現在の実装: 墓守の死亡時点で死んでいた味方のみ対象）

### 17. ガーディアン「攻撃をすべて引き受ける」
- 味方にガーディアンが2体いる場合はどうなる？
- シャドウウォーカーのランダム攻撃もガーディアンが引き受けるか？（現在の実装: シャドウウォーカーはガーディアンを無視してランダム）

### 18. オーク「攻撃時、相手にピヨリを付与」
- ピヨリの付与は100%か？確率か？（現在の実装: 100%付与、行動不可の発動は30%確率）

### 19. ゾンビ「前後の味方に毒攻撃を付与」
- ゾンビ自身も毒攻撃を持つか？（現在の実装: 持つ）
- 毒の付与確率は？（現在の実装: 30%）

## バトルシステム全般の疑問

### 20. 範囲攻撃
- 仕様書に「範囲攻撃：対象とその前後にダメージ」とあるが、現在範囲攻撃を持つ魔物がいない。将来追加予定か？

### 21. ターン上限
- バトルにターン上限はあるか？双方回復手段があると無限に続く可能性がある

### 22. 状態異常の持続ターン
- 毒・ピヨリ・魔法バリアの持続は永続か？ターン数制限があるか？（現在の実装: 全て永続）

### 23. 同速時の行動順
- 同じ速度の場合の処理順は？（現在の実装: 前衛側が先に行動）

### 24. 2枠ユニットへの攻撃
- 「どちらの枠への攻撃も受ける」とあるが、1回の攻撃でダメージは1回？2回？（現在の実装: 1回）
