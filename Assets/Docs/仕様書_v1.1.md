# Bits Bite Back 仕様書 v1.1

## ゲーム概要
- ジャンル：ローグライト × 隊列オートバトル
- プレイ人数：1人
- 対戦形式：PvE（AI敵との連戦）
- 想定プレイ時間：1プレイ15〜30分（モック）

---

## プレイヤー
- 人数：1人
- 初期状態：長い骨×3, 頭蓋骨(動物)×1, 長い木×2 + チュートリアル用スケルトン1体 + 魂3個（活力D, 猛攻D, 迅速C）
- 勝利条件：ダンジョンの各階層のボスを倒して次の階層へ進む
- 敗北条件：隊列の魔物が全滅する

## 練成システム

### 基本練成
- 素材を組み合わせて魔物を練成する
- 使う素材の種類と量によって練成できる魔物が決まる
- ガラクタを多く使うほど、同じ魔物でもランクが高くなりやすい
- 練成時に触媒を使うとS/A/Bランクが出やすくなる
- 練成された魔物の年数は0で始まる

### 分解
- 不要な魔物を分解して素材を回収する
- 分解で得られる素材は練成に使った素材の一部
- 分解時に魂を1つ獲得する（年数に応じたランクの魂がランダムな種類で生成）
- 分解確認ダイアログで返却素材・獲得魂ランクのプレビューを表示
- 分解完了ダイアログで獲得素材・獲得魂ランクを表示

### 触媒
- 宝箱から15%の確率でドロップする
- 練成時に使用するとS/A/Bランクの出現率が上がる
- フッターの「触媒」タブで所持数を確認可能

### モックでの制限事項
- 素材による見た目変化は未実装（後日追加予定）

## 年数システム

- 各魔物は「年数」パラメータを持つ（age）
- 練成時に年数0で生まれる
- ダンジョン内で1歩移動するたびに、隊列の全魔物の年数が+1される
- カード表示・隊列スロットに「N年」と表示される
- 年数は分解時の魂ランク決定に使用される

---

## 魂システム

### 魂の種類（5種類）

| 名前 | SoulType | 効果 |
|---|---|---|
| 活力の魂 | Vigor | HP +効果値 |
| 猛攻の魂 | Fury | ATK +効果値 |
| 迅速の魂 | Swiftness | SPD +効果値 |
| 剛健の魂 | Fortitude | HP +効果値/2, ATK +効果値/2（複合） |
| 精密の魂 | Precision | 射程 +1（Bランク以上は+2） |

### ランクと効果値

| ランク | 効果値 |
|---|---|
| D | 3 |
| C | 6 |
| B | 10 |
| A | 15 |
| S | 20 |

### 魂の獲得
- 分解時に魂を1つ獲得する
- 魂の種類はランダム、ランクは分解する魔物の年数で決定

| 年数 | 魂ランク |
|---|---|
| 0〜19年 | D |
| 20〜49年 | C |
| 50〜99年 | B |
| 100〜199年 | A |
| 200年以上 | S |

### 魂の装備
- 1体の魔物に1個まで装備可能
- 装備を外して別の魔物に付け替えられる
- バトル時はResetToBaseで装備ボーナスが自動適用される
- ステータス表示は全て装備込み（GetEffectiveStats）

### 魂の管理
- MaterialInventory の List<SoulData> で管理
- 初期状態で魂3個所持（活力D, 猛攻D, 迅速C）

## フェーズ構成（1プレイ）

```
フェーズ1：準備（練成・分解・隊列編成）
フェーズ2：ダンジョン探索（移動・宝箱・イベント・エンカウント）
  ├── シンボルエンカウント → バトル（勝利後ダンジョンに戻る、素材報酬あり）
  ├── 宝箱 → 素材入手
  └── イベント → 回復 等
フェーズ3：ボス戦（フロア最奥のボスとバトル）
→ ボス撃破 → 3択報酬選択 → 次の階層の準備画面へ
→ 敗北でタイトルに戻る
```

> 準備はダンジョンに入る前に1回のみ。ダンジョン内では練成・編成できない。

---

## 階層システム

- ボス撃破で次の階層へ進む（CurrentFloor が1ずつ増加）
- 階層が進むと部屋数・敵数が増加し、難易度が上がる
- 3階層ごとに敵の質が変わり体感が変化する（スケルトン級→毒・射程→回復・タンク→フル編成）
- 階層クリア報酬は3択: 全回復 / 素材(3+floor個) / バランス型(HP50%回復+素材少し)
  - 報酬選択はバトル画面内で行い、選択後に準備画面へ遷移する

## ダンジョン探索システム

### マップ生成
- ランダム生成（部屋＋通路の構造）
- 各階層ごとに1フロア構成。ボス撃破で次の階層へ
- マップサイズは25×25固定。部屋数・敵数は階層に応じて増加
- 未探索部分はフォグで隠れる
- ミニマップを画面に表示し、探索済み部分が追加表示される

### 移動
- ターン制グリッド移動（1マスずつ、方向キーまたはWASD）
- 見下ろしビュー（2Dドット絵テイスト）
- プレイヤーが1マス移動するたびに敵シンボルも確率で1マスランダム移動する
- 敵シンボルがプレイヤーに接触した場合もエンカウントが発生する

### シンボルエンカウント
- 敵シンボルがダンジョン内に配置される
- プレイヤーが敵シンボルに接触（または敵がプレイヤーに到達）→ BattleScene に遷移
- バトル勝利 → 素材報酬（2 + floor/2 個）を獲得 → ダンジョンに戻る（敵シンボル消滅）
- バトル敗北 → タイトルに戻る

### 宝箱
- ダンジョン内にランダム配置される（各部屋に最大1つ）
- 中身は素材（ランダム）。素材量は階層が進むと増加する
- 15%の確率で触媒が追加でドロップする
- 一度取得した宝箱は消える

### イベント
- ダンジョン内にイベントマスがランダム配置される（「?」黄色で表示）
- イベントがある部屋にはイベント以外のエンティティを配置しない（部屋を専有）
- イベントマスに移動すると、30%の確率で敵が出現する（Weak 60% / Medium 40%）
- 敵が出現しない場合は回復イベント（味方全体のHPを最大HPの30%回復）
- 敵出現確率はDungeonConfigSO.eventEnemyChanceで調整可能
- 後からイベントの種類を追加可能（罠・商人・選択肢付きイベント等）

### ボス
- フロア最奥にボス部屋がある
- ボス部屋に入る → 確認ダイアログ →「挑む」で BattleScene に遷移
- ボスの編成は階層によって変化する（階層1はスケルトン×2、階層が進むとガーディアン+α等）
- ボス撃破 → 3択報酬選択 → 次の階層の準備画面へ

---

## バトルシステム

### 基本ルール
- ターン制オートバトル
- 速度が速い魔物から順番に行動する
- 隊列は最大5枠（2枠ユニットあり）

### 攻撃射程
- 射程内で一番近い敵を攻撃する（生存位置ベースの判定、attackerPos + targetPos < range）
- 貫通攻撃：射程内の敵すべてにダメージ
- 範囲攻撃：対象とその前後にダメージ

### 編成位置
- 前衛：攻撃しやすいがダメージも受けやすい
- 後衛：安全だが射程が短いと攻撃できない
- 2枠ユニット：編成枠を2つ使用し、どちらの枠への攻撃も受ける

### 状態異常
| 名前 | 効果 |
|---|---|
| 毒 | ターン開始時にHP-3 |
| ピヨリ | 行動時に確率で行動不可 |
| 魔法バリア | ダメージ半減（魔法攻撃に対して） |

### バトル終了条件
- 勝利：敵の魔物を全滅させる
- 敗北：味方の魔物が全滅する

## 画面操作フロー

### 準備画面（PrepareScene）

画面はタブ切替式。上部に階層情報とタブボタン（編成/練成/分解/所持魂）、下部に所持素材（素材/触媒タブ切替）と出撃ボタン。

**編成タブ：**
1. 上部に隊列5枠が横並びで表示される（前衛←→後衛）
2. 下部に所持魔物がグリッド表示される
3. 魔物カードに年数（N年）・装備中の魂を表示
4. ステータスは装備込みの値（GetEffectiveStats）を表示
5. 魔物カードに「魂装備」「魂を外す」ボタンを表示
6. 魔物カードをクリックして選択（黄色い枠でハイライト）
7. 隊列スロットをクリックして配置（2枠ユニットは隣接枠も使用）
8. 配置済みスロットをクリックすると隊列から外す

**練成タブ：**
1. 練成可能な魔物がグリッド表示される（左に画像、右に名前・パラメータ・能力・必要素材）
2. カードをクリック → 練成確認ダイアログ（通常練成 / 触媒で練成 / キャンセル）
3. 練成完了ダイアログで結果を表示（ランク・ステータス）

**分解タブ：**
1. 隊列外の魔物がグリッド表示される（練成と同じカード構成 + 返却素材）
2. カードをクリック → 分解確認ダイアログ（魔物画像+ステータス+返却素材+獲得魂ランク予告）
3. 分解完了ダイアログで獲得素材・獲得魂ランクを表示

**所持魂タブ：**
1. 所持している魂がグリッド表示される（左:種類色オーブ、右:名前[ランク]+効果）
2. ランク高い順 → 種類順でソートされる

**魂装備フロー：**
1. 編成タブの魔物カードから「魂装備」ボタンをクリック
2. 魂選択ダイアログ（グリッド表示、装備可能な魂のみ）
3. 魂を選択 → 装備確認ダイアログ（ステータス変化プレビュー）
4. 「装備する」で装備完了

**フッター：**
- 素材タブ：所持素材の一覧
- 触媒タブ：所持触媒の数

**出撃：**
- 隊列に1体以上配置した状態で「出撃！」ボタンをクリック → ダンジョン画面へ

### ダンジョン画面（DungeonScene）

画面中央にダンジョンマップ（見下ろしビュー）、右上にミニマップ、上部にHP等のステータス。

**探索中：**
1. 方向キー（またはスワイプ/クリック）で1マスずつ移動
2. 移動するたびに敵シンボルも移動する（ターン制）
3. 未探索エリアはフォグで隠れている
4. ミニマップに探索済みエリアが表示される

**シンボルエンカウント：**
- 敵シンボルに接触 → バトル画面に遷移
- バトル勝利 → ダンジョンに戻る

**宝箱：**
- 宝箱マスに移動 → 素材を自動入手 → メッセージ表示

**イベント：**
- イベントマスに移動 → 30%で敵出現（バトル遷移）/ 70%で回復イベント → メッセージ表示

**ボス部屋：**
- ボス部屋に入る → 確認メッセージ →「挑む」で BattleScene に遷移

### バトル画面（BattleScene）

画面は上部にヘッダー（ターン/Wave）、中央に敵（左）と味方（右）の隊列（前衛同士が中央で対峙）、下部にバトルログ。

**バトル中：**
- オートバトルが自動進行する（操作不要）
- 右下の「速度」ボタンでバトル速度を切り替え（1x / 2x / 5x / 10x の4段階）
- 「スキップ」ボタンで即座にバトル結果を出す（演出なしで高速決着）
- バトルログで各ユニットの行動を確認できる

**バトル終了後：**
- 通常バトル勝利 →「続ける」→ ダンジョンに戻る（素材報酬を自動獲得）
- ボス戦勝利 → 3択報酬選択パネル表示 → 選択後に次の階層の準備画面へ
- 敗北 →「続ける」→ タイトル画面に戻る
