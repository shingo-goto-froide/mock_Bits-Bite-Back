# Bits Bite Back 仕様書 v1.1

## ゲーム概要
- ジャンル：ローグライト × 隊列オートバトル
- プレイ人数：1人
- 対戦形式：PvE（AI敵との連戦）
- 想定プレイ時間：1プレイ15〜30分（モック）

---

## プレイヤー
- 人数：1人
- 初期状態：基本素材一式 + チュートリアル用魔物1体
- 勝利条件：ダンジョンのボスを倒す
- 敗北条件：隊列の魔物が全滅する

## 練成システム

### 基本練成
- 素材を組み合わせて魔物を練成する
- 使う素材の種類と量によって練成できる魔物が決まる
- ガラクタを多く使うほど、同じ魔物でもランクが高くなりやすい

### 分解
- 不要な魔物を分解して素材を回収する
- 分解で得られる素材は練成に使った素材の一部

### モックでの制限事項
- 魂システムは未実装（後日追加予定）
- 素材による見た目変化は未実装（後日追加予定）

## フェーズ構成（1プレイ）

```
フェーズ1：準備（練成・分解・隊列編成）
フェーズ2：ダンジョン探索（移動・宝箱・イベント・エンカウント）
  ├── シンボルエンカウント → バトル（勝利後ダンジョンに戻る）
  ├── 宝箱 → 素材入手
  └── イベント → 回復 等
フェーズ3：ボス戦（フロア最奥のボスとバトル）
→ 勝利でクリア / 敗北でタイトルに戻る
```

> 準備はダンジョンに入る前に1回のみ。ダンジョン内では練成・編成できない。

## ダンジョン探索システム

### マップ生成
- ランダム生成（部屋＋通路の構造）
- 1フロア構成（後から階層追加が可能な設計）
- 未探索部分はフォグで隠れる
- ミニマップを画面に表示し、探索済み部分が追加表示される

### 移動
- ターン制グリッド移動（1マスずつ）
- 見下ろしビュー（2Dドット絵テイスト）
- プレイヤーが1マス移動するたびに敵シンボルも1マス移動する

### シンボルエンカウント
- 敵シンボルがダンジョン内に配置される
- プレイヤーが敵シンボルに接触 → BattleScene に遷移
- バトル勝利 → ダンジョンに戻る（敵シンボル消滅）
- バトル敗北 → タイトルに戻る

### 宝箱
- ダンジョン内にランダム配置される
- 中身は素材（ランダム）
- 一度取得した宝箱は消える

### イベント
- ダンジョン内にイベントマスがランダム配置される
- 現在実装するイベント：**回復**（味方全体のHPを一定量回復）
- 後からイベントの種類を追加可能（罠・商人・選択肢付きイベント等）

### ボス
- フロア最奥にボス部屋がある
- ボス部屋に入る → ボス戦（BattleScene に遷移）
- モックではオークをボスとして配置
- ボス撃破 → ダンジョンクリア

---

## バトルシステム

### 基本ルール
- ターン制オートバトル
- 速度が速い魔物から順番に行動する
- 隊列は最大5枠（2枠ユニットあり）

### 攻撃射程
- 射程内で一番遠い敵を攻撃する
- 貫通攻撃：射程内の敵すべてにダメージ
- 範囲攻撃：対象とその前後にダメージ

### 編成位置
- 前衛：攻撃しやすいがダメージも受けやすい
- 後衛：安全だが射程が短いと攻撃できない
- 2枠ユニット：編成枠を2つ使用し、どちらの枠への攻撃も受ける

### 状態異常
| 名前 | 効果 |
|---|---|
| 毒 | ターン開始時にHP-3 |
| ピヨリ | 行動時に確率で行動不可 |
| 魔法バリア | ダメージ半減（魔法攻撃に対して） |

### バトル終了条件
- 勝利：敵の魔物を全滅させる
- 敗北：味方の魔物が全滅する

## 画面操作フロー

### 準備画面（PrepareScene）

画面はタブ切替式。上部にWave情報とタブボタン（編成/練成/分解）、下部に所持素材と出撃ボタン。

**編成タブ：**
1. 上部に隊列5枠が横並びで表示される（前衛←→後衛）
2. 下部に所持魔物がグリッド表示される
3. 魔物カードをクリックして選択（黄色い枠でハイライト）
4. 隊列スロットをクリックして配置（2枠ユニットは隣接枠も使用）
5. 配置済みスロットをクリックすると隊列から外す

**練成タブ：**
1. 練成可能な魔物がグリッド表示される（左に画像、右に名前・パラメータ・能力・必要素材）
2. カードをクリックすると素材を消費して練成される

**分解タブ：**
1. 隊列外の魔物がグリッド表示される（練成と同じカード構成 + 返却素材）
2. カードをクリックすると分解され素材が戻る

**出撃：**
- 隊列に1体以上配置した状態で「出撃！」ボタンをクリック → ダンジョン画面へ

### ダンジョン画面（DungeonScene）

画面中央にダンジョンマップ（見下ろしビュー）、右上にミニマップ、上部にHP等のステータス。

**探索中：**
1. 方向キー（またはスワイプ/クリック）で1マスずつ移動
2. 移動するたびに敵シンボルも移動する（ターン制）
3. 未探索エリアはフォグで隠れている
4. ミニマップに探索済みエリアが表示される

**シンボルエンカウント：**
- 敵シンボルに接触 → バトル画面に遷移
- バトル勝利 → ダンジョンに戻る

**宝箱：**
- 宝箱マスに移動 → 素材を自動入手 → メッセージ表示

**イベント：**
- イベントマスに移動 → イベント発動（回復等）→ メッセージ表示

**ボス部屋：**
- ボス部屋に入る → 確認メッセージ →「挑む」で BattleScene に遷移

### バトル画面（BattleScene）

画面は上部にヘッダー（ターン/Wave）、中央に敵（左）と味方（右）の隊列（前衛同士が中央で対峙）、下部にバトルログ。

**バトル中：**
- オートバトルが自動進行する（操作不要）
- 右下の「速度」ボタンでバトル速度を切り替え（1x / 2x / 5x）
- バトルログで各ユニットの行動を確認できる

**バトル終了後：**
- 通常バトル勝利 →「続ける」→ ダンジョンに戻る
- ボス戦勝利 →「次へ」→ クリア演出 → タイトル画面に戻る
- 敗北 →「続ける」→ タイトル画面に戻る

## パラメータ一覧

| パラメータ名 | 値 | 説明 |
|---|---|---|
| 最大隊列枠数 | 5 | 2枠ユニットは2枠消費 |
| ピヨリ発動確率 | 30% | 行動不可になる確率 |
| 毒ダメージ | 3 | ターン開始時の固定ダメージ |
| 2連射開始ターン | 5 | アーチャーの2連射が始まるターン |
| ダンジョンマップサイズ | 調整可能 | ランダム生成のフロアサイズ |
| 敵シンボル数 | 調整可能 | フロア内の敵シンボル配置数 |
| 宝箱数 | 調整可能 | フロア内の宝箱配置数 |
| イベント数 | 調整可能 | フロア内のイベントマス数 |
| 回復イベント回復量 | 調整可能 | 味方全体のHP回復量 |
