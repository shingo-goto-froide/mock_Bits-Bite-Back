# Bits Bite Back シーン構成書 v1.1

## シーン一覧

| シーン名 | 役割 | Build Index | パス |
|---|---|---|---|
| TitleScene | タイトル・ゲーム開始 | 0 | Assets/Scenes/TitleScene.unity |
| PrepareScene | 準備（練成・分解・隊列編成） | 1 | Assets/Scenes/PrepareScene.unity |
| BattleScene | オートバトル・報酬表示 | 2 | Assets/Scenes/BattleScene.unity |

```
TitleScene
```

## PrepareScene

### Hierarchy

```
PrepareScene
├── EventSystem
├── Main Camera
└── PrepareCanvas
    ├── WaveBar                              ← Wave表示（上5%）
    │   └── WaveText
    ├── TabButtonPanel                       ← タブボタン（5%）
    │   ├── FormationTabButton "編成"
    │   ├── CraftTabButton "練成"
    │   └── DisassembleTabButton "分解"
    ├── FormationTabContent                  ← 編成タブ（75%）
    │   ├── FormationPanel（上20%）
    │   │   ├── FormationTitle "隊列"
    │   │   ├── FormationDirection "前衛 ← → 後衛"
    │   │   └── SlotsParent
    │   │       ├── FormationSlot_0
    │   │       ├── FormationSlot_1
    │   │       ├── FormationSlot_2
    │   │       ├── FormationSlot_3
    │   │       └── FormationSlot_4
    │   └── MonsterListPanel（下80%）
    │       ├── MonsterListTitle "所持魔物"
    │       └── MonsterScrollView
    │           └── Viewport
    │               └── Content             ← MonsterCardUIを動的生成
    ├── CraftTabContent（初期非表示）        ← 練成タブ
    │   └── CraftPanel
    │       ├── CraftTitle "練成可能な魔物"
    │       └── CraftScrollView
    │           └── Viewport
    │               └── Content             ← 練成可能な魔物を動的表示
    ├── DisassembleTabContent（初期非表示）  ← 分解タブ
    │   └── DisassemblePanel
    │       ├── DisassembleTitle "分解（隊列外の魔物）"
    │       └── DisassembleScrollView
    │           └── Viewport
    │               └── Content             ← 分解対象を動的表示
    └── FooterPanel                          ← フッター（15%）
        ├── HeaderPanel（所持素材バー）
        │   ├── MaterialLabel "所持素材"
        │   └── MaterialScrollView
        │       └── Viewport
        │           └── MaterialPanel
        │               ├── MaterialText_AnimalSkull
        │               ├── MaterialText_HumanSkull
        │               ├── MaterialText_LongWood
        │               ├── MaterialText_LongBone
        │               └── MaterialText_OldSword
        ├── BattleStartButton "出撃！"（中央）
        └── MessageText
```

### コンポーネント一覧

| GameObject | コンポーネント | 備考 |
|---|---|---|
| EventSystem | EventSystem, InputSystemUIInputModule | StandaloneInputModuleは使わない |
| Main Camera | Camera | ClearFlags=SolidColor |
| PrepareCanvas | Canvas(ScreenSpace-Overlay), CanvasScaler(1920×1080), GraphicRaycaster, PrepareUI | |
| TabButtonPanel | HorizontalLayoutGroup | タブ切替ボタン横並び |
| SlotsParent | HorizontalLayoutGroup | 編成スロット横並び |
| FormationSlot_0〜4 | FormationSlotUI, Button, Image, Outline | |
| MonsterScrollView/Content | GridLayoutGroup(3列, 580×200) | |
| CraftScrollView/Content | GridLayoutGroup(3列, 580×200) | |
| DisassembleScrollView/Content | GridLayoutGroup(3列, 580×200) | |
| MaterialPanel | HorizontalLayoutGroup, ContentSizeFitter | 素材アイコン横並び |
| BattleStartButton | Button, TextMeshProUGUI(子) | "出撃！" |

### Inspector参照設定

PrepareUIは全てFindDeepで名前検索するため、Inspector参照設定なし。

## BattleScene

### Hierarchy

```
BattleScene
├── EventSystem
├── Main Camera
├── [BattleManager]
└── [BattleCanvas]
    ├── [BattleHeader]
    │   ├── TurnText
    │   └── WaveText
    ├── [EnemyFormation]                    ← 左側(0%〜48%)、敵は右が前衛（味方側）
    │   ├── DirectionLabel "後衛 ← → 前衛"
    │   └── EnemyContent
    │       └── (BattleUnitView × N を動的生成)
    ├── [PlayerFormation]                   ← 右側(52%〜100%)、味方は左が前衛
    │   ├── DirectionLabel "前衛 ← → 後衛"
    │   └── PlayerContent
    │       └── (BattleUnitView × N を動的生成)
    ├── [BattleLogPanel]
    │   ├── LogScrollView
    │   │   └── LogContent
    │   │       └── LogText
    │   └── SpeedButton
    ├── [ResultPanel] ← 初期非表示
    │   ├── ResultText
    │   └── ContinueButton
    └── [RewardPanel] ← 初期非表示
        ├── RewardTitle
        ├── RewardScrollView
        │   └── Viewport
        │       └── Content
        └── RewardContinueButton
```
