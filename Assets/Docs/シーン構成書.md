## DungeonScene

> マップ描画はSpriteRendererベース（Tilemap不使用）。DungeonUIがBuildMap()で動的に"DungeonTiles"を生成。

### Hierarchy

```
DungeonScene
├── EventSystem
├── Main Camera（Orthographic, Size=5, プレイヤー追従用）
├── [DungeonManager]（Singleton）
├── DungeonTiles（動的生成、タイル・フォグのSpriteRenderer群の親）
├── Player（動的生成、先頭魔物スプライト + 緑枠）
├── Entity_Enemy_*（動的生成、敵シンボル）
├── Entity_Treasure_*（動的生成、宝箱）
├── Entity_Event_*（動的生成、イベント "?" 黄色）
└── [DungeonCanvas]
    ├── [HudPanel]（上部HUD）
    │   ├── PartyHpPanel（隊列HPバー表示、HorizontalLayoutGroup）
    │   ├── MaterialText（所持素材表示、TextMeshProUGUI）
    │   └── TurnText（ターン表示、TextMeshProUGUI）
    ├── [MinimapPanel]（右上ミニマップ）
    │   └── MinimapImage（RawImage）
    ├── [MessagePanel]（下部メッセージ、初期非表示）
    │   └── MessageText（TextMeshProUGUI）
    └── [BossConfirmPanel]（ボス戦確認、初期非表示）
        ├── BossConfirmText（TextMeshProUGUI "ボスに挑みますか？"）
        └── ButtonContainer
            ├── BossConfirmButton（Button "挑む"）
            └── BossCancelButton（Button "戻る"）
```

### コンポーネント一覧

| GameObject | コンポーネント | 備考 |
|---|---|---|
| EventSystem | EventSystem, InputSystemUIInputModule | StandaloneInputModuleは使わない |
| Main Camera | Camera(Orthographic, Size=5) | ClearFlags=SolidColor(暗い色), Z=-10 |
| [DungeonManager] | DungeonManager (Singleton) | config は Resources.Load で自動取得。Awakeで階層に応じてconfigを上書き |
| DungeonTiles | Transform（動的生成） | タイル・フォグの SpriteRenderer 群の親。シーン遷移時に分割破棄 |
| Player | SpriteRenderer（動的生成） | 先頭魔物スプライト + 緑枠（AddOutline） |
| [DungeonCanvas] | Canvas(ScreenSpace-Overlay), CanvasScaler(1920×1080), GraphicRaycaster, DungeonUI | |
| [HudPanel] | HorizontalLayoutGroup, Image(背景) | 上部10%、半透明背景 |
| PartyHpPanel | HorizontalLayoutGroup, LayoutElement(flexibleWidth=3) | 動的にHP表示を生成 |
| MaterialText | TextMeshProUGUI, LayoutElement(flexibleWidth=2) | "素材なし" 初期値 |
| TurnText | TextMeshProUGUI, LayoutElement(preferredWidth=120) | "階層 1  ターン 0" 初期値 |
| [MinimapPanel] | Image(背景), MinimapUI | 右上固定、アンカー(0.85-0.99, 0.7-0.89) |
| MinimapImage | RawImage | ミニマップテクスチャ表示 |
| [MessagePanel] | Image(背景), CanvasGroup | 下部中央、初期非表示 |
| MessageText | TextMeshProUGUI | メッセージ表示用 |
| [BossConfirmPanel] | Image(背景), CanvasGroup, VerticalLayoutGroup | 中央、初期非表示 |
| BossConfirmText | TextMeshProUGUI | "ボスに挑みますか？" |
| BossConfirmButton | Button, Image | "挑む" |
| BossCancelButton | Button, Image | "戻る" |

## BattleScene

### Hierarchy

```
BattleScene
├── EventSystem
├── Main Camera
├── [BattleManager]
└── [BattleCanvas]
    ├── [BattleHeader]
    │   ├── TurnText
    │   └── WaveText
    ├── [EnemyFormation]                    ← 左側(0%〜48%)、敵は右が前衛（味方側）
    │   ├── DirectionLabel "後衛 ← → 前衛"
    │   └── EnemyContent
    │       └── (BattleUnitView × N を動的生成)
    ├── [PlayerFormation]                   ← 右側(52%〜100%)、味方は左が前衛
    │   ├── DirectionLabel "前衛 ← → 後衛"
    │   └── PlayerContent
    │       └── (BattleUnitView × N を動的生成)
    ├── [BattleLogPanel]
    │   ├── LogScrollView
    │   │   └── LogContent
    │   │       └── LogText
    │   ├── SkipButton（動的生成、SpeedButtonの上に配置）
    │   └── SpeedButton（速度: 1x / 2x / 5x / 10x の4段階）
    ├── [ResultPanel] ← 初期非表示
    │   ├── ResultText
    │   └── ContinueButton
    ├── [RewardPanel] ← 初期非表示（通常Wave用）
    │   ├── RewardTitle
    │   ├── RewardScrollView
    │   │   └── Viewport
    │   │       └── Content
    │   └── RewardContinueButton
    └── [FloorRewardPanel] ← ボス撃破時に動的生成
        ├── RewardOverlay（半透明黒背景）
        └── RewardPanel
            ├── Title "階層 N クリア！ 報酬を選んでください"
            └── RewardRow（HorizontalLayout、3択ボタン）
                ├── RewardBtn "全回復"
                ├── RewardBtn "素材×N"
                └── RewardBtn "回復+素材"
```
