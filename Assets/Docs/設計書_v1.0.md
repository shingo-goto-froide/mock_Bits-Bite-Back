## 難易度曲線

モックは全7ウェーブの連戦で構成する。

| ウェーブ | 段階 | 敵構成 | 目的 |
|---|---|---|---|
| 1 | 序盤（学習） | スケルトン×2 | バトルの流れを理解する |
| 2 | 序盤（学習） | ゾンビ×1, スケルトン×2 | 状態異常（毒）の存在を知る |
| 3 | 中盤（挑戦） | ガーディアン×1, アーチャー×2 | タンク＋後衛の連携に対処する |
| 4 | 中盤（挑戦） | 骸骨僧侶×1, ガーディアン×1, ゾンビ×1, スケルトン×1 | 回復＋タンクのシナジーを崩す |
| 5 | 中盤（転換点） | 結界師×1, ガーディアン×1, アーチャー×1, スケルトン×1, ゾンビ×1 | 本格的なシナジー編成との対決 |
| 6 | 終盤（ボス） | オーク×1(2枠), 骸骨僧侶×1, 結界師×1 | 高ステータスのボス＋サポート |
| 7 | 終盤（最終） | リベンジャー×1, レイス×2, 墓守×1 | 死亡シナジー編成。逆転が起きる |

- 序盤（Wave1-2）：基本操作を覚える。失敗しても素材は得られ、練成で立て直せる
- 中盤（Wave3-5）：隊列の組み方で明確に差が出る。シナジーの重要性を体感する
- 終盤（Wave6-7）：強敵に対して自分の戦術が通用するか試す。逆転の快感

## フィードバックループ

### 短いループ（数秒：1アクション）
- 攻撃 → ダメージ表示 → HPバー減少
- 状態異常付与 → アイコン表示
- 特殊効果発動 → 演出＋テキスト通知

### 中くらいのループ（1ターン）
- ターン開始 → 毒ダメージ → 行動順に各ユニット行動 → ターン終了
- 隊列の配置が結果に直結する（前衛が先に倒れる、射程が届かない等）

### 長いループ（1バトル〜1プレイ）
- バトル勝利 → 報酬（素材） → 新しい魔物を練成 → 次のバトルで試す
- 全滅 → 「あの編成ならいけたかも」→ 準備からやり直し
- 企画書の編成例3パターンを自力で発見・再現する楽しさ

## バランス数値の設計方針

### ダメージ計算
```
基本ダメージ = 攻撃者の攻撃力
魔法バリアあり（魔法攻撃時）: × 0.5
ガーディアン（魔法攻撃被弾時）: × 1.5
最終ダメージ = 基本ダメージ（小数点以下切り捨て、最低1）
```
- 防御力は設けない（モックのシンプルさ優先）
- 攻撃力 = そのままダメージ。直感的で調整しやすい

### 速度（行動順決定用）

仕様書に速度の記載がないため、キャラクターの性格に基づき設定する。
同速の場合は先に前衛側が行動する。

| 魔物 | 速度 | 根拠 |
|---|---|---|
| シャドウウォーカー | 5 | 暗殺者。最速で奇襲 |
| アーチャー | 4 | 遠距離から先制 |
| シェイプシフター | 4 | 素早く変身して行動 |
| レイス | 4 | 幽体。素早い |
| スケルトン | 3 | 標準的な速さ |
| 結界師 | 3 | バリアは開始時なので速度は中程度 |
| リベンジャー | 3 | 標準。仲間の死で強化されてから動く |
| ゾンビ | 2 | 遅い。毒は付与済みなので問題なし |
| 骸骨僧侶 | 2 | 回復は戦闘後なので速度は低くてよい |
| 殉教者 | 2 | 味方のダメージを見てから回復 |
| 墓守 | 2 | 遅め。死亡時効果なので速度不要 |
| ガーディアン | 1 | 重装。遅いがタンクなので問題なし |
| オーク | 1 | 重い。ただし高火力で確実に殴る |

### 調整の方向性
- **攻撃力を上げる** → バトルが短くなる。爽快感は上がるが戦術性が下がる
- **HPを上げる** → バトルが長くなる。シナジーが発動しやすくなるが冗長になりうる
- **速度を変える** → 行動順が変わりシナジーの発動タイミングが変化。バランスへの影響大
- **毒ダメージを変える** → 長期戦の有利不利に影響
- **ピヨリ確率を変える** → 高すぎると理不尽、低すぎると存在感がない

## リスク・リワード設計

### 攻め vs 守りの構図
企画書の編成例がそのままリスク・リワードの3段階を表現している：

| 編成例 | 戦略 | リスク | リワード |
|---|---|---|---|
| ①守りの隊列 | ガーディアンで守り、毒でジワジワ削る | 低リスク：安定するが強化要素がない | 低リワード：後半は火力不足になる |
| ②攻めの隊列 | シェイプシフター＋オークで高火力 | 中リスク：シェイプシフターが倒れると火力半減 | 高リワード：バトルを重ねるほどオークが強化 |
| ③逆転の隊列 | 味方を犠牲にしてリベンジャーを強化 | 高リスク：死ぬ順番がズレると瓦解 | 最高リワード：成功すればHP3倍・攻撃力5倍の貫通攻撃 |

### 逆転要素
- リベンジャー：味方が死ぬほど強くなる → 劣勢からの逆転
- 墓守：死亡時に全味方復活 → 壊滅からの立て直し
- アーチャー：5ターン目以降に2連射 → 長期戦になるほど有利
